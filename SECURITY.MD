## Security Reporting

If you have Identified a potential security vulnerability, please use GitHub's private reporting feature to report the vulnerability directly to us. **Do not disclose any privately reported vulnerabilies until the vulnerability is triaged/disclosed to the public by us.**

The following do not count as security vulnerabilites:

- Security issues in 3rd party packages that we use (report the issue to them instead of us)
- Issues with Client Components (due to the nature of roblox, it is not possible to provide 100% security on the client)
